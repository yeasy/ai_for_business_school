# 附录

正文回答“怎么想、怎么做”，附录负责“随手可查、拿来即用”。本附录共八个文件，按用途可分三类：**开会带的**——工具速查与项目清单，供立项、选型、汇报现场直接使用；**课堂用的**——讨论题与课程地图，供商学院教学与企业读书会组织研讨；**查证用的**——证据索引、参考文献、术语表与延伸阅读，供引用数据前核对口径、读后深挖来源。

| 附录 | 内容 | 适用场景 |
|---|---|---|
| [术语表](glossary.md) | 全书关键术语的简明定义，按主题分组并标注出处章节 | 阅读中随手查；帮团队统一语言 |
| [决策工具与框架速查](toolkit.md) | 正文框架的可打印版：场景决策 2×2 模板、起步五步法检查单、供应商五问与合同三底线、数据就绪四维自查、向董事会汇报一页纸等 | 开会带：立项评审、供应商谈判、董事会汇报现场使用 |
| [开源项目与平台清单](projects.md) | 正文提及的开源框架、协议与商业平台一览 | 开会带：选型调研的起点，配合 6.3 与 10.4 使用 |
| [数据与案例证据索引](evidence_index.md) | 全书关键数据与案例的口径、出处，按 ✅✅/✅/⚠️ 三级可信度标注，并说明分级标准 | 查证：对外引用任何数字前，先来这里核对口径 |
| [分章讨论题](discussion.md) | 十二章逐章讨论题，重点包括 Klarna 复盘、2×2 场景决策、“优化还是重构”课堂辩论 | 课堂用：案例研讨、读书会、企业内训 |
| [课程地图：12 周与 16 周方案](teaching_guide.md) | 12 周方案逐周对应十二章；16 周方案增加案例研讨与期末项目 | 课堂用：教师排课与教学设计 |
| [延伸阅读](further_reading.md) | 按主题推荐的书目、报告与一手资料 | 查证与深读：读完某章想继续深入时 |
| [参考文献](references.md) | 全书引用来源的完整汇总 | 查证：溯源与学术引用 |

一点使用建议：管理者优先把[决策工具与框架速查](toolkit.md)打印出来放在手边；师生请从[课程地图](teaching_guide.md)与[分章讨论题](discussion.md)入手组织课程；而无论哪类读者，转述书中任何数据之前，请养成翻一次[证据索引](evidence_index.md)的习惯——分清“研究显示”与“事实如此”，正是本书想传递的思维方式。
