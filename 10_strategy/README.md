# 第十章 战略选择与投资节奏

前九章回答了智能体“是什么、为什么是现在、如何落地”；从本章起，视角转向领导者本人。本章回答一个问题：在 AI 议题上，一把手究竟该亲自做哪几个判断、按什么节奏投入资源。

全章五节层层递进。10.1 从一个反直觉的判断开场：AI 的第一个用户应当是一把手自己——智能体第一次让最高决策者绕过层层修饰的信息链，直接看到未经处理的业务数据，这也是全书最希望读者带走的记忆点之一。10.2 讨论如何把“我们要不要上 AI”这个采购问题，换成两个真正的战略问题：哪些环节正在变得十倍便宜，以及这对双方护城河意味着什么。10.3 给出三大战略选项——防御、延伸、颠覆，分别以银行业、电商平台与 AI 制药为锚点案例，并用资源基础观（VRIN）论证专有数据、判断力与信任为何构成新护城河。10.4 提供一张 2×2 场景决策矩阵，回答每个具体场景该买现成、该主导掌控、该暂缓还是该小注验证。10.5 讨论投入节奏——可逆的小额多下注、不可逆的大额等信号——以及如何向董事会与资本市场讲这笔账。

本章的分工边界如下：10.4 只回答需求侧的战略判断（这个场景落哪一格、要不要自建），供应商如何评估、合同如何谈判见 [6.3 选型与谈判](../06_ecosystem/6.3_sourcing.md)；算账框架与预算量级属第七章（[7.3](../07_value/7.3_cost_benefit.md)、[7.4](../07_value/7.4_budget.md)），10.5 只引用其口径而不重复展开；叫停与复盘的操作纪律在 [9.6](../09_landing/9.6_exit_discipline.md)，10.5 讲的是如何把叫停纳入对董事会的组合叙事；10.1 与 [9.4 起步五步法](../09_landing/9.4_five_steps.md)的分工，是领导者个人使用习惯与组织落地方法之别。
