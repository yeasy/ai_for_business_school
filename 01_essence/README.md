# 第一章 人工智能的商业本质

人工智能领域几乎每周都有新进展，但企业决策需要的不是新闻，而是一个足够稳定的认知框架。本章回答四个问题：这波浪潮的本质是什么？大模型在经济学上属于哪一类技术？它今天能做什么、不能做什么？管理者最容易在哪些判断上出错？

四节依次展开。1.1 节用"三级跳"给出全景：感知智能"会看"、生成智能"会写"、行动智能"会干活"——前两级改变的是机器有多聪明，只有第三级触动企业的成本结构与组织结构。1.2 节把大模型放进经济史的坐标系：它具备通用目的技术的全部特征，与蒸汽机、电力同类；电气化"生产率悖论"的百年教训，将预先解释第九章那个刺眼的"95%"。1.3 节画出能力边界的商业地图：边界不在任务多复杂，而在流程能否理清、结果能否核验。1.4 节逐一拆解五个最常见的认知误区，给出分辨"跟风"与"真落地"的三个信号，并埋下贯穿全书的题眼——行业经验和数据是 AI 时代最稀缺的入场券。

分工边界需要事先说明一次——全书都按这套"同一议题分层展开、各章各司其职"的方式组织，此处交代清楚，后续各章不再重复。本书从四个层面讨论"AI 能力边界"：本章 1.3 只回答商业层现象，即"能与不能"的直观图景；任务判据的操作化细化在 2.4；错误背后的技术成因在 4.3；值不值得做的经济判据在 7.5。同样，1.2 只讨论大模型的经济属性，技术机理与局限由第四章展开。本章原则上不引用具体案例数字、不展开落地方法——证据在第八章，方法在第九章。

读完本章，读者应能用自己的语言回答：为什么智能体不是又一次工具升级，以及在铺天盖地的 AI 叙事中，如何一眼分辨谁在讲故事、谁在真干。
