# 第八章 行业落地案例深读

第七章给出了算账的框架，本章把框架放到事实上检验。全章回答三个问题：各行业的领先者到底做成了什么；那些广为流传的数字，各自的统计口径是什么；对于规模不及巨头的多数企业，其中哪些做法可以复制。

全章六节。8.1 至 8.4 依次深读制造与基建、金融、零售与电商、医药与生命科学四个行业，每节末尾固定一段面向腰部企业的翻译——“如果你是腰部企业，最先卡在哪一步”，把标杆案例的做法拆到多数企业够得着的动作，并点明其中最容易卡壳的一步。8.5 用整节篇幅完整复盘 Klarna 的 AI 客服案例——从账面辉煌，到质量反转，再到人机协同收场——这是全书唯一一个把“上篇与下篇”讲全的案例。8.6 换一个横切视角：同样的技术，巨头、腰部与中小企业的正确打法完全不同，规模决定路径。

本章所有案例遵循统一模板：**背景—做法—指标—口径与局限**。前三项交代事实，第四项交代数字是怎么统计出来的、在什么范围内成立。之所以把“口径与局限”写进模板，是因为 AI 领域的宣传数字格外密集：厂商愿景被当作实测、行业平均被当作个体承诺、相关性被当作因果、单点最佳被当作平台平均。学会读 AI 统计数字，本身就是本章的教学目的之一。阅读时建议对每个数字保持三个追问：分子分母是什么、参照系是什么、是实测还是愿景。

分工边界：本章只讲“发生了什么、数字怎么读”。为什么多数试点没有回报、如何从试点走到规模化，是第九章的主题；由案例引出的战略选择与自建采购决策，在第十章展开；案例中涉及的组织与人才问题，在第十一章展开。本章各节在相应位置给出指引，不重复展开。
